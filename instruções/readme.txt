Aplicação web

Requisitos para utilização em localhost

-Utilizar xampp, 

-O projeto deve ficar dentro da pasta "htdocs" do xampp.

-Baixar o projeto público no github https://github.com/AndradeCCS/paper-2023-1,

-Baixar o banco modelo na pasta "estruturaBanco",

-Criar o banco de dados utilizando a ferramenta do xampp e o nome do banco de dados como "cadastropacientes". Caso de outro nome ao banco de dados será necessário ajustar o arquivo config.php e ajustar o parametro de conexão informando o banco que foi  criado  "define('BASE', 'cadastropacientes');",

O usuário de login é admin e a senha é 123. casoqueria criar um novo usuário ele deve ser criado diretamente no banco de dados na tabela usuários.

OBS: o arquivo style.css foi retirado do github do professor Gustavo Guanabara, segue link para download do arquivo https://github.com/gustavoguanabara/php-moderno/tree/main/downloads/modelo-css.
